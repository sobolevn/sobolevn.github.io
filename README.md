# sobolevn's personal page

[See it live!](https://sobolevn.me)


## Development

Make sure you have `ruby 2.5.1` installed, use `rbenv` in case you don't.

1. `rbenv install`
2. `gem install bundler`

Then run:

1. `bundle install`
2. `bundle run jekyll s --safe --strict_front_matter`
3. Done!
